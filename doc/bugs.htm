Lack of Menu Stack

  • BRIEF: Menu lacks a stack thus causing navigational glitches and problems.
  • AFFECTS: The entire menu system.
  • CAUSE: When a new menu is loaded the current menu is set and all records of a previous menu are wiped, the only way to revert is to use the explicit previous menu.
  • DETAILED DESCRIPTION: Navigating through the menus and going back and forth is a huge mess, there is no way to easily do this kind of navigation. Also if a message appears over another message the game is stuck in a menu.
  • HOW TO FIX: Remove explicit previous menu and instead use a stack so that navigation going back and forth can easily be done.

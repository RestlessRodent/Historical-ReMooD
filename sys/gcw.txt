Controls:
 - SELECT: Open Menu

When menu is visible:
 - START: Select Item
 - UP: Move selector up
 - DOWN: Move selector down
 - LEFT: Decrease value
 - RIGHT: Increase value

When playing/spectating:
 - UP: Move Forwards
 - DOWN: Move Backwards
 - LEFT: Turn Left
 - RIGHT: Turn Right
 - Y: Attack
 - X: Speed
 - A: Use/Join Game
 - L: Strafe Left
 - R: Strafe Right
 - START: Use Inventory Item
 - B+Y: Land
 - B+A: Jump
 - B+L: Previous Weapon
 - B+R: Next Weapon
 - B+START: Chat
 - B+X+UP: Look Down
 - B+X+DOWN: Look Up
 - B+X+Y: Automap
 - B+X+A: Coop-Spy
 - B+X+L: Previous Inventory
 - B+X+R: Next Inventory
 - B+X+START: Suicide
